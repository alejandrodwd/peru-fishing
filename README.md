# Spatiotemporal Patterns of Industrial Fishing Pressure Near Peru’s EEZ Boundary

In this brief investigation into industrial fishing along Peru's EEZ I: searched for, cleaned, validated, merged and analyzed a public dataset to draw conservative conculusions from patterns found in the data. To handle all the data, I mainly used geopandas and matplotlib, and other supplementary python libraries like pycountry and shapely. The final results and commentary on it can be found in /notebooks/exploration.ipynb.
